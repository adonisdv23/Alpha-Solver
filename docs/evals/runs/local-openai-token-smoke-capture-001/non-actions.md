# Non-actions

The lane did not:

- use private data;
- send secrets or credentials;
- send hidden instructions;
- send raw sensitive evidence;
- send private operator notes;
- send customer data or private business data;
- send unredacted logs;
- run evals or benchmarks;
- compare Alpha Solver against another provider;
- deploy anything;
- expose `/v1/solve` or dashboards;
- modify runtime behavior, provider routing behavior, model/provider code, tests, or CI;
- update Google Sheets;
- print, store, or commit API keys;
- record private billing details.
