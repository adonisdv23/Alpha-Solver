# Codex usage guidance

Use one prompt for one selected Level 10 lane. Do not combine lanes unless a controlling packet explicitly says to do so. Before editing, verify the Level 9 plan acceptance, gateway status, current-main-based branch, and allowed changed-file scope.

After editing, run the lane-required checks, confirm changed-file scope, commit, and prepare a PR body that repeats the evidence boundary and explicit non-actions.
