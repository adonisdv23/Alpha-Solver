# Review Task Set

Use this small representative set for display inspection only. Do not use it to score model behavior or generate quality claims.

| Task ID | Prompt shape | What the route panel should make inspectable |
|---|---|---|
| CRMP-001 | Short deterministic math/checking prompt. | Basic recommended mode/model and no-call route preview language. |
| CRMP-002 | Longer synthesis prompt with context-pressure language. | Context tier, fallback candidates, grouped reasons, and warnings. |
| CRMP-003 | Privacy-sensitive prompt that should remain local or constrained. | Privacy tier, backend tier, warnings, and evidence-boundary wording. |
| CRMP-004 | Tool-looking prompt that asks for file/web/GitHub action. | Tool recommendation preview and execution authorization status. |
| CRMP-005 | Unsupported/no-eligible-route scenario. | Fail-closed preview, no eligible route messaging, and non-execution language. |
