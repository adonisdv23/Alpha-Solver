# Non-Actions

This lane does not:

- Run the console manually.
- Perform the operator review.
- Run Alpha runtime behavior or invoke `/v1/solve`.
- Call providers, run hosted models, run local models, or run Ollama.
- Execute tools for task answering, browse, make runtime GitHub calls, or mutate Sheets.
- Score, unblind, inspect source maps, deploy, or inspect raw source identities.
- Add helpers, dependencies, runtime behavior, API behavior, routing behavior, or product-test automation.
