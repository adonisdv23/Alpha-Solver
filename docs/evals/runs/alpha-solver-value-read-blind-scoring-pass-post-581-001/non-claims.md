# Non-Claims

This scoring-only lane does not support or make claims about:

- Readiness.
- Value.
- Provider behavior or validation.
- Local-model behavior or validation.
- Security/privacy completion.
- Production readiness.
- Public readiness.
- Dashboard readiness.
- `/v1/solve` readiness.
- Benchmark proof or benchmark superiority.
- Partnership status.
- Pi.dev integration.
- Final interpretation.
- Alpha superiority.
