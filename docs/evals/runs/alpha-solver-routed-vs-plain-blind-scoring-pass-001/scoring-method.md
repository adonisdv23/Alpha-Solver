# Scoring method

- Scorer method: Codex blind rubric scorer.
- Materials used: approved scorer-facing packet plus authorization/prep packet only.
- Scale: frozen 0 to 3 rubric dimensions from the approved rubric.
- Procedure: scored Response A and Response B independently by task and dimension, then recorded blind preference, rationale, caveat, contested-score flag, scorer method, and timestamp.
- Boundary: no unblinding, source identity review, source artifact review, route metadata review, source-map review, A/B key review, external research, runtime execution, provider call, local-model call, tool execution, or final interpretation.
