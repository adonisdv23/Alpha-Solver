# Stop Conditions

Stop immediately and return to operator review if any of the following occur:

- Scorer identity or method is ambiguous.
- Scorer packet path is missing.
- Score output path is missing.
- Unblinding is requested.
- Final interpretation is requested.
- Provider, local model, runtime, dashboard, public API, `/v1/solve`, Google Sheets, dependency, routing, council, or benchmark action is requested without explicit separate authorization.
- Identity inference, identity-map access, identity reconstruction, or source attribution is requested.
