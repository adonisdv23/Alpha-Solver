# Non-Actions

This lane does not and must not:

- Score outputs or fill blind scores.
- Invoke a scorer.
- Unblind, inspect, reconstruct, or commit an identity map.
- Create final interpretation.
- Call providers, use provider tokens, access credentials, or inspect billing.
- Run hosted models or local models.
- Pull, install, or add models or dependencies.
- Expose runtime behavior, dashboard behavior, public API behavior, or `/v1/solve`.
- Modify runtime, API, dashboard, routing, council, benchmark, or scoring product code.
- Mutate Google Sheets or external backlog ledgers.
