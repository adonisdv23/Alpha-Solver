# Non-Actions

This lane does not:

- score outputs
- fill scores
- choose winners
- compute aggregate totals
- unblind
- interpret results
- inspect or commit A/B key
- inspect or commit source map
- inspect source artifacts for scoring
- run Alpha runtime
- invoke `/v1/solve`
- call providers
- run hosted or local models
- execute tools
- browse or use current external research
- mutate Google Sheets
- add dependencies
- expose dashboards or public APIs
- deploy
