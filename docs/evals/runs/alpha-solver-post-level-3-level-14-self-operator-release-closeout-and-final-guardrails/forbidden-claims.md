# Forbidden claims

The following phrases are forbidden as project status claims in this closeout packet:

- MVP ready
- release ready
- production ready
- runtime ready
- provider ready
- hosted ready
- benchmark validated
- autonomous ready
- broad user ready

The packet must also not use provider/model/API/browser/deployment/billing/credential references as closeout capability claims. Those terms may appear only in boundary documentation, forbidden-claim documentation, or scan results.
