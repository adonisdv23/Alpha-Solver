# Smoke-Test Task Set

This task set is a future placeholder for a default-skipped local smoke test. It is not an executed result.

## Minimum future task

1. Build a local adapter request from the portable contract and one short operator-approved user prompt.
2. Send the request only to the operator-approved localhost endpoint.
3. Use the exact operator-supplied model name.
4. Apply a finite timeout.
5. Preserve raw request/response artifacts in the future execution lane.
6. Import only sanitized summaries after execution, if execution occurs.

## Prohibited task expansion

The future smoke task must not expand into runtime routing, public solve route, dashboard preview, hosted service fallback, benchmark, billing, Batch C, or provider-orchestration work.
