# Release blockers

Self Operator MVP remains blocked from release, runtime exposure, or evidence promotion until all of the following are separately authorized and satisfied:

- static test scaffold is implemented and reviewed;
- runtime behavior is specified in a later explicit runtime lane;
- operator approval records and stop states are implemented and tested;
- artifact persistence is implemented without forbidden artifact content;
- route exposure, deployment, billing, provider use, fallback, and evidence promotion remain blocked unless separately authorized.
