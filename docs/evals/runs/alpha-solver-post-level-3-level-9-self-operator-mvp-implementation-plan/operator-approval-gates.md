# Operator approval gates

A future implementation lane must be operator-supervised. It must stop unless:

- this Level 9 implementation plan has been accepted;
- the selected Level 10 lane matches the controlling packet;
- the branch is current-main-based;
- allowed scope is static test scaffold only;
- changed files are allowed;
- raw artifacts and reviewer notes will be captured.

Operator approval does not waive hard boundaries for provider calls, external API calls, credentials, browser automation, deployment, billing, route exposure, fallback, or evidence promotion.
