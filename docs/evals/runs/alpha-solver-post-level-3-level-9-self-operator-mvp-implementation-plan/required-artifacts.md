# Required artifacts

Future implementation runs must preserve raw artifacts and reviewer notes. Required artifacts include:

- command transcript or command summary for checks run;
- staged and unstaged diff proof;
- changed-file scope proof;
- static test output;
- raw artifact inventory;
- reviewer notes explaining pass/fail evidence;
- stop-state record if any blocked condition appears.

Artifacts must remain local-only and must not contain secrets, provider outputs, external API responses, billing data, browser data, deployment output, or evidence-promotion labels.
