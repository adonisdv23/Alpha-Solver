# Stop Conditions

Stop immediately if any of the following occur:

- live pre-edit verification fails;
- the mechanism is no longer manual no-provider prompt-contract simulation;
- provider, token, credential, billing, hosted-model, local-model, model-pull, or model-install access would be needed;
- product/runtime code, `/v1/solve`, dashboard, public API, or Google Sheets would be used;
- data outside committed synthetic packet material would be required;
- raw Alpha or raw baseline output paths are missing or ambiguous;
- the selected pilot subset changes without operator authorization;
- scoring, blind-score filling, unblinding, final interpretation, benchmark behavior, or value/readiness claims are requested;
- an operator-only unblinding map would be created or committed;
- runtime, API, dashboard, routing, council, benchmark, or scoring behavior would need modification.
