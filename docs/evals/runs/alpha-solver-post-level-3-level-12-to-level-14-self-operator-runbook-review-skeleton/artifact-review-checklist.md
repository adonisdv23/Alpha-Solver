# Artifact Review Checklist

- [ ] Schema version present.
- [ ] Run ID present.
- [ ] Lane ID present.
- [ ] Operator confirmation recorded.
- [ ] Stop state recorded when applicable.
- [ ] Findings are local-only and operator-supervised.
- [ ] Artifact paths are inside the allowed local output directory.
- [ ] Redaction status is recorded.
- [ ] Evidence boundary is stated.
- [ ] No unredacted secrets.
- [ ] No credentials.
- [ ] No provider outputs or hosted model outputs.
- [ ] No external API responses.
- [ ] No browser, deployment, billing, or Google Sheets data.
- [ ] No source-artifact mutation.
- [ ] No evidence promotion beyond the local-only boundary.
