# Allowed inspection scope

Future lanes may inspect the following local repository materials only as needed to keep the first-code scope narrow:

- Level 8 Self Operator planning and stop-condition packets.
- Level 7 local run harness design packet.
- Level 9 first-code scope contract packet.
- Existing docs that describe entrypoint roles and operator workflows.
- Existing test organization and fixture naming patterns.
- Existing checker command availability.

Inspection must remain local and offline. It must not fetch remotes, install packages, call networks, call providers, read credentials, open browsers, deploy, bill, or run models.
