# Copy/Paste Artifact Separation Rule

- Never combine separate copy/paste artifacts in one block.
- ED, GS, PR comments, Codex prompts, runbook commands, and operator messages must each be placed in separate blocks.
- If multiple artifacts are needed, label them separately and keep each copy/paste block self-contained.
- Do not append unrelated roadmap text inside a copy/paste artifact.
- Do not combine merge decision text with GS update text.
