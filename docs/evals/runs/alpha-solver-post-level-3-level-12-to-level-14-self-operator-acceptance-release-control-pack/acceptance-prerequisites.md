# Acceptance Prerequisites

Manual local acceptance must not run until all gates are satisfied:

1. Level 10 static-test scaffold merged and GS done.
2. Level 11 local artifact/preflight foundation merged and GS done.
3. Level 11 approval/stop-state/gate foundation merged and GS done.
4. Level 12 local dry-run harness wrapper merged and GS done.
5. Final runbook reviewed.
6. Explicit operator confirmation present; stop if explicit operator confirmation is missing.
7. Evidence boundary reviewed.
8. Output artifact path confirmed local-only.
9. No provider/API/dashboard/CLI/browser/deployment/billing/credential/Google Sheets/source-artifact/evidence-promotion scope.
