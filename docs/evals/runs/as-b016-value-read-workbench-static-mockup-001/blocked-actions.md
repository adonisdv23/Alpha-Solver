# Blocked Actions

B016 blocks and does not perform:

- runtime code changes
- provider execution changes
- Operator Console runtime behavior changes
- webapp route changes
- live UI behavior
- new routes or POST routes
- real-run cockpit creation
- provider calls
- hosted or local model calls
- `/v1/solve` exposure or invocation
- output scoring
- score mutation
- unblinding
- source identity reveal
- final interpretation
- Google Sheets or external ledger mutation
- fake outputs, fake scores, fake benchmark numbers, or fake source identities
- readiness, value, benchmark, production, public-readiness, security/privacy, provider-validation, local-model-validation, or Alpha-superiority claims
