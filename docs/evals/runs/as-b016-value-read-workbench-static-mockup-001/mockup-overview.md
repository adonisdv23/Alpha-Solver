# Mockup Overview

B016 converts the B015 text-only first-screen wireframe into a static Markdown mockup. The mockup is source-truth review material only.

## Review questions

- What am I reviewing?
- Is it complete?
- What can I safely do next?
- What can I not claim?

## Boundaries

The mockup does not read live files, parse packets, call providers, run models, expose `/v1/solve`, create routes, score outputs, unblind results, reveal source identities, perform final interpretation, or mutate external ledgers.
