# Forbidden claims

This packet must not be used to claim any of the following:

- OpenAI validation.
- Provider validation.
- API smoke success.
- Token usage.
- Runtime readiness.
- Production readiness.
- Public MVP readiness.
- Security/privacy completion.
- Benchmark validation.
- Benchmark superiority.
- Broad-user readiness.
- Autonomous readiness.
- `/v1/solve` readiness.
- Dashboard readiness.
- Billing evidence or private billing readiness.

Claims remain blocked until separately authorized and evidenced.
