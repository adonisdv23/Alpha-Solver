Future acceptance and release closeout must proceed in this order:

1. Finish static-test scaffold and GS.
2. Finish artifact schema and persistence scaffold and GS.
3. Finish local preflight runner and command classification guardrails and GS.
4. Finish approval and stop-state scaffolds and GS.
5. Finish dry-run harness wrapper and GS.
6. Prepare acceptance packet.
7. Run operator-supervised local-only acceptance.
8. Import raw/redacted results.
9. Interpret acceptance results.
10. Update MVP runbook.
11. Run boundary review.
12. Produce release closeout packet.

This file defines sequencing only. It does not claim any acceptance step has started or completed.
