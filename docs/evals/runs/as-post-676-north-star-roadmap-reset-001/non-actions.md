# Non-Actions

This lane does not:

- Implement runtime code.
- Modify provider execution.
- Modify Operator Console runtime behavior.
- Modify webapp routes.
- Add UI behavior.
- Add routes or POST routes.
- Create a real-run cockpit.
- Call providers.
- Run hosted models or local models.
- Expose `/v1/solve`.
- Create action controls.
- Create a generic LLM playground.
- Add browser automation.
- Add CLI/subprocess execution from the web UI.
- Add shell execution.
- Add a queue, runner, scheduler, worker, or background job.
- Add a new write path.
- Change scores, scoring, unblinding, final interpretation, or source identity handling.
- Mutate Google Sheets or backlog workbooks.
