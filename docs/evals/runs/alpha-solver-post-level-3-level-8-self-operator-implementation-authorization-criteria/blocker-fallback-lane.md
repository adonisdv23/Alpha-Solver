# Blocker Fallback Lane

Blocker fallback lane:

`ALPHA-SOLVER-POST-LEVEL-3-LEVEL-8-SELF-OPERATOR-IMPLEMENTATION-`
`AUTHORIZATION-CRITERIA-FIX-001`

## Use when

Use this fallback lane if any required file is missing, any authorization criterion is ambiguous, any required boundary is incomplete, any check fails, or a future implementation proposal cannot prove all criteria before modifying Self Operator runtime code.

## Boundary

The fallback lane is for criteria repair only. It does not authorize implementation, runtime modification, provider calls, browser automation, credentials, deployment, billing, autonomous merges, `/v1/solve` exposure, dashboard exposure, or evidence promotion.
