# Blocked Claims

The accepted result must not be used to claim or imply any of the following:

- local model quality
- benchmark evidence
- production readiness
- MVP readiness
- provider-orchestration evidence
- Alpha superiority
- billing evidence
- dashboard readiness
- `/v1/solve` readiness
- broad runtime readiness
- evidence-model promotion
- product readiness
- broad runtime validation
- hosted provider evidence
- provider fallback readiness
- hosted fallback readiness

The accepted result must not be used to authorize provider fallback, hosted fallback, dashboard exposure, `/v1/solve` exposure, product readiness claims, benchmark claims, or evidence promotion.
