# Fable audit source boundary

The final independent Fable audit and post-#489 strategy audit were same-thread external chat audit sources, not committed raw repo artifacts.

This annex records their findings as derivative audit references only. It does not convert those chat reports into raw execution evidence, does not add raw transcripts, and does not mutate prior source evidence.

Source classification used by this annex:

- `same_thread_fable_chat_report`: derivative audit reference from the same external chat thread.
- `repo_artifact_found`: committed repository artifact observed in the synchronized post-#490 checkout.

Council readers should separate derivative audit references from committed raw execution artifacts when weighing evidence.
