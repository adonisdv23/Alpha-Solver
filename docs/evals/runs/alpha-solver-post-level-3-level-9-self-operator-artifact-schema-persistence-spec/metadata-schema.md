# Metadata schema

Required metadata fields:

- `schema_version`
- `lane_id`
- `run_id`
- `created_at_utc`
- `repo_branch`
- `commit_sha_before_run`
- `local_only: true`
- `operator_supervised: true`
- `evidence_promotion: false`
- `provider_calls: false`
- `external_api_calls: false`
- `credentials_used: false`
- `browser_automation: false`
- `deployment: false`
- `billing: false`
