# Redaction markers

Use explicit markers when content is intentionally excluded:

- `[REDACTED_SECRET_BLOCKED]`
- `[REDACTED_CREDENTIAL_BLOCKED]`
- `[REDACTED_PROVIDER_OUTPUT_BLOCKED]`
- `[REDACTED_EXTERNAL_API_RESPONSE_BLOCKED]`
- `[REDACTED_BILLING_DATA_BLOCKED]`
- `[REDACTED_BROWSER_DATA_BLOCKED]`
- `[REDACTED_DEPLOYMENT_OUTPUT_BLOCKED]`

Redaction markers must describe blocked categories without preserving sensitive content.
