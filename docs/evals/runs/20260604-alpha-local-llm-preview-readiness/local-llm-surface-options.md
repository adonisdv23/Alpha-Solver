## Surface conclusion

The safest next technical path is not to use the current local preview as behavior evidence. If implementation is approved, start with explicit local LLM provider/contract-consumption proof, then decide whether `/dashboard/expert-preview`, `/v1/solve`, or a dev-only route should expose it.
