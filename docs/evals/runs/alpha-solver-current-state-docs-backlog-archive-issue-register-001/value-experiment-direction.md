# Value experiment direction

Full direction doc: [`docs/VALUE_EXPERIMENT_DIRECTION.md`](../../../VALUE_EXPERIMENT_DIRECTION.md).

- A tiny OpenAI smoke proves **plumbing only** — never Alpha Solver value.
- The next strategic validation is **`ALPHA-SOLVER-VALUE-EXPERIMENT-PROTOCOL-001`**.
- It should compare Alpha Solver vs a **plain baseline**, on enough tasks to
  reduce noise, with **verbosity/brevity controls**, **no self-referential
  governance prompts**, **blind scoring**, and **pre-registered pass/fail
  thresholds**.
- **Not run in this lane.** No provider call, token, or eval. No benchmark or
  superiority claim.
