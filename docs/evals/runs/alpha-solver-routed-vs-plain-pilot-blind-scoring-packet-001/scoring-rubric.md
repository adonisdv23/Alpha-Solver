# Scoring rubric for later blind scoring

This rubric is for a later separately authorized blind scoring pass only. Do not fill scores in this lane.

Score each answer independently from 0 to 3 for each dimension:

| Score | Meaning |
|---|---|
| 0 | Missing, unsafe, nonresponsive, or materially violates the task boundary. |
| 1 | Partially responsive with major gaps, weak constraint handling, or unclear usefulness. |
| 2 | Mostly responsive and useful with minor gaps or caveats. |
| 3 | Strong, direct, constraint-preserving, useful, and appropriately bounded. |

Dimensions for later scoring:

1. Direct answer quality
2. Constraint following
3. Task-specific correctness or usefulness
4. Evidence-boundary preservation
5. Uncertainty handling
6. Safety/high-stakes handling
7. Hallucination resistance
8. Route/structure usefulness without revealing route identity
9. Actionability
10. Concision/readability

No scores, winners, totals, or interpretations are recorded here.
