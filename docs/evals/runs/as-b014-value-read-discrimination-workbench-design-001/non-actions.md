# Non-actions

This lane does not:

- implement runtime code
- modify provider execution
- modify Operator Console runtime behavior
- modify webapp routes
- add UI behavior
- add routes or POST routes
- create a real-run cockpit
- call providers
- run hosted or local models
- expose or invoke `/v1/solve`
- score outputs
- unblind results
- reveal source identities
- perform final interpretation
- mutate Google Sheets or external ledgers
- add browser automation
- add web UI CLI/subprocess/shell execution
- add queues, runners, schedulers, workers, or background jobs
- authorize B012 or B013
- formalize Roadmap HP as a repo module
