# Non-Actions and Evidence Boundary

## Evidence boundary

This is a docs-only future runbook draft. It does not implement Self Operator, run Self Operator, run models, call providers, deploy, expose routes, or promote evidence.

## Non-actions

This packet does not:

- create, modify, or validate a Self Operator implementation;
- define a runnable command, API route, scheduler, job queue, provider adapter, credential flow, or deployment path;
- run local models, hosted models, provider APIs, fallback providers, or evaluation suites;
- call external providers or incur provider spend;
- deploy, expose routes, change runtime configuration, or alter environment expectations;
- update backlog workbooks or external ledgers;
- change source code, tests, configs, specs, or existing evidence packets;
- promote any artifact into readiness, safety, quality, performance, benchmark, provider, or production evidence.

## Blocked claims

Do not claim from this packet that Self Operator:

- exists;
- is implemented;
- is ready;
- is safe to run;
- has an approved runtime path;
- can call providers safely;
- can manage credentials safely;
- can recover from stops safely;
- has generated valid artifacts;
- is ready for MVP, preview, production, benchmark, or evidence promotion.
