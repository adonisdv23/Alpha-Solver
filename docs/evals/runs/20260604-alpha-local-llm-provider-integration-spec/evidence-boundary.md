# Evidence Boundary

## Allowed evidence from this lane

This lane may be cited only as specification evidence for a future packet. It
selects one future provider shape and records authorization, test, failure,
rollback, and non-goal requirements.

## Labels for future work

- `offline_fixture_evidence`: static parser/request checks with no service
  contact.
- `local_smoke_evidence`: local endpoint smoke output only if separately
  authorized later.
- `non_evidence_wiring`: seam construction and metadata-preservation checks.
- `failed_closed_result`: normalized failure outcomes.

## Not evidence for

This lane is not evidence for local model execution, Ollama execution,
hosted-provider execution, API route readiness, dashboard preview readiness,
runtime readiness, MVP status, production status, Alpha quality, comparative
Alpha claims, broad plain-provider comparison, Batch C status, benchmark
outcomes, exact billing outcomes, or provider orchestration.
