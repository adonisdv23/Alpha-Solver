# Redaction checklist template

- [ ] Prompt is synthetic or sanitized.
- [ ] No secrets.
- [ ] No private keys.
- [ ] No credentials.
- [ ] No private operator notes.
- [ ] No raw sensitive evidence.
- [ ] No customer/private business data.
- [ ] No hidden instructions.
- [ ] No raw logs containing secrets.
- [ ] Only explicitly selected files are referenced.
- [ ] Output reviewed before commit.
- [ ] Sensitive output stop condition not triggered.
