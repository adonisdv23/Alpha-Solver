- runtime smoke execution;
- local model quality evidence;
- hosted provider evidence;
- `/v1/solve` readiness;
- dashboard readiness;
- MVP validation;
- production readiness;
- benchmark evidence;
- provider orchestration evidence;
- Alpha superiority evidence;
- evidence-model promotion;
- broad runtime readiness evidence.

Confidence returned by Pass 1 is a local orchestration signal only. It is not local model quality evidence, benchmark evidence, production-readiness evidence, MVP validation, or evidence-model promotion.
