# Synthetic prompt boundary

The first smoke lane may use only narrow, synthetic, non-sensitive prompts.

## Allowed prompt types

- Synthetic Alpha Solver task.
- Public-style non-sensitive task.
- Short task with no private data.
- No repo secrets.
- No private evidence.
- No hidden instructions.

## Forbidden prompt types and contents

- Secrets.
- Credentials.
- Customer data.
- Private business data.
- Raw logs.
- Private operator notes.
- Hidden chain-of-thought requests.
- Unredacted evidence packets.
