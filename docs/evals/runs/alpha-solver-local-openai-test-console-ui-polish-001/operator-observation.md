# Operator observation

The Operator reviewed the local-only test console and noted usability gaps:

- The page still looked like a plain developer HTML form.
- Mode was a dropdown but model was a plain text field.
- Both mode and model should be dropdowns.
- Very long prompts could fail without a clear operator-facing explanation.
- The result reason and errors were not clear enough.
- The app should feel like an actual local testing console.

These are presentation and usability observations. They are not behavior, quality, or readiness findings.
